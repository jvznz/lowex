Como configurar o Google Agenda ▼
Passo 1 — Criar projeto no Google Cloud
Acesse console.cloud.google.com → crie um projeto novo → ative a Google Calendar API.
Passo 2 — Criar credencial OAuth
Em "Credenciais" → "Criar credencial" → "ID do cliente OAuth 2.0" → tipo Aplicativo da Web.
Em "Origens JavaScript autorizadas" adicione: http://localhost e o domínio do seu site.
Passo 3 — Tela de consentimento
Configure a tela OAuth (pode ser externo + modo de teste). Adicione seu e-mail como usuário de teste.
Passo 4 — Cole o Client ID aqui
Copie o "ID do cliente" (termina em .apps.googleusercontent.com) e cole no campo acima → Salvar → Conectar.
✓ Tudo acontece no seu browser — nenhum dado sai para servidor externo. O token OAuth fica salvo localmente.
